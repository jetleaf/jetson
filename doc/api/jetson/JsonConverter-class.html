JsonConverter

JsonConverter<T extends JsonSerializable> class

Annotation used to associate a custom JSON converter with a class field or property.

This is especially useful when a field cannot be serialized or deserialized using the default behavior and needs a specific JsonSerializable implementation.

The converter should point to a class that extends JsonSerializable<T>.

Example:

class MyModel {
  @JsonConverter(type: ClassType<DateTimeAdapter>(null, "package:jetleaf_lang/lang.dart"))
  final DateTime createdAt;

  MyModel(this.createdAt);
}

When deserializing MyModel, the DateTimeAdapter will be used to convert the createdAt field instead of the default strategy.

This annotation is part of the Jetleaf Reflection system.

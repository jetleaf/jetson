isNumber method

  1. @override
bool isNumber()
override

Returns true if this node represents a JSON number.

JSON numbers can be either int or double in Dart. This method ensures that the node can safely be cast to num before arithmetic operations.

Example

if (node.isNumber()) {
  final number = node.toObject() as num;
  print(number + 10);
}

Implementation

@override
bool isNumber() => true;

Sets a custom JsonGenerator for the object mapper being built.

This allows users to override the default JSON generator used for serialization. Once set, the provided generator will be used whenever the object mapper serializes Dart objects to JSON strings or maps.

This is part of the builder pattern, so the method returns the builder itself to allow fluent chaining of configuration calls.

Example

final mapper = JetsonObjectMapperBuilder()
    .jsonGenerator(MyCustomJsonGenerator())
    .build();

Implementation

JetsonObjectMapperBuilder jsonGenerator(JsonGenerator jsonGenerator) {
    _jsonGenerator = jsonGenerator;
    return this;
}
